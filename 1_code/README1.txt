---- Encrypted Chat Software ----
HOW TO USE:

This software needs to use Java RuntimeEnvironment. 
All class should be compiled together using java compiler. 
You need to prepare one computer as server and the others as client.

Client:
Client needs to connect to the network
Client runs ChatClient.java
Connect to your server using IP address that have been provided by server

Server:
You need to run ChatServer.java.
Makesure all your clients are connected to server.
